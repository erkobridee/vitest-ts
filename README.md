# vitest-ts

Vitest TypeScript project template

## Commands

### Creation

```
npm init

npm i -D \
  @types/jest \
  @types/node \
  esbuild \
  typescript \
  vitest \
  vite-tsconfig-paths
```

### Development

The following command will start the vitest

```
npm start

or

npm run dev
```

### Create a new project using this as a template

```
npx degit erkobridee/vitest-ts ${project_name}
```
